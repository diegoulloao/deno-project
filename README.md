# API Rest with Deno

This is a Deno test implementing a simple API Rest application

# Serving Locally
```sh
deno run --allow-net index.ts
```

# Serving from Remote
```sh
deno run --allow-net https://raw.githubusercontent.com/diegoulloao/deno-project/master/index.ts
```
